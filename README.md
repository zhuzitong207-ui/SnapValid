# SnapValid
一款轻量强大的**键盘鼠标自动化 + OCR验证码识别 + 截图相似度校验**工具，专注于自动化验证场景，开箱即用、易扩展。

## ✨ 核心特性
- 🎯 键盘鼠标全操作录制/回放（精准到毫秒级）
- 🔍 OCR验证码自动识别（支持多语言、自定义模板）
- 🖼️ 区域截图相似度校验（支持多阈值、多屏适配）
- 🖥️ 多屏复制模式自动检测（适配多显示器场景）
- 🌐 浏览器自动最大化（兼容Chrome/Edge/Firefox）
- 📝 详细日志记录 + 结构化结果输出（便于问题排查）
- 📦 绿色单文件打包（无环境依赖，直接运行）
- ⚙️ 灵活配置（支持自定义快捷键、OCR参数）

## ⌨️ 快捷键说明
| 快捷键          | 功能描述                 |
|-----------------|--------------------------|
| F9              | 开始录制操作             |
| F10             | 回放已录制的脚本         |
| F11             | 停止录制/回放            |
| Ctrl+Alt+I      | 插入OCR识别步骤          |
| Ctrl+Alt+S      | 插入截图相似度校验步骤   |
| Ctrl+Alt+R      | 重置所有录制状态         |
